# Rex.js 轻量级无构建前端框架，专为快速原型开发设计，让想法快速落地

> 一个轻量级无构建前端框架，专为快速原型开发设计。零构建依赖、内置响应式系统、模板指令、组件化、路由与 API 请求能力，让原型开发更高效。

## 核心特点

✅ **零构建依赖** - 直接引入 JS 文件即可使用，无需 webpack/vite/rollup 等构建工具  
✅ **极致轻量** - 核心代码精简无冗余，加载速度快，无额外依赖负担  
✅ **原生响应式** - 内置 ref/reactive 响应式系统，数据驱动视图，告别手动 DOM 操作  
✅ **模板驱动开发** - 指令式语法直观易懂，快速实现页面交互逻辑  
✅ **灵活组件化** - 支持模板/脚本两种组件定义方式，封装复用更高效  
✅ **内置基础路由** - 满足多页面原型需求，无需额外引入路由库  
✅ **便捷 API 请求** - 内置接口请求指令，快速对接后端数据，无需手动封装请求


## 快速开始

- 现代浏览器（Chrome ≥ 79、Firefox ≥ 72、Edge ≥ 79、Safari ≥ 14）
- 无需 Node.js 环境，纯前端运行
